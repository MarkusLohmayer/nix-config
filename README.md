# nix-config

Nix flake containing
- [nix-darwin](https://github.com/LnL7/nix-darwin) system configuration
- [home-manager](https://github.com/nix-community/home-manager) user configuration


## Installation

```
sh <(curl -L https://nixos.org/nix/install)
./switch darwin
./switch m_one_home
```
